📦 1.1 变量与数据类型（最基础的基础）

💡

什么是变量？

变量就是给数据起一个名字，方便以后使用。就像给快递箱子贴标签——"这箱是张三的"，你就知道里面是什么了。
在Python里，不需要声明类型，直接写 名字 = 值 就行。

1

字符串（str）—— 存文字内容

字符串就是文本，用引号（单引号或双引号都行）括起来。

python
# 最基础的字符串 name = "张三" company = '半亩花田' # 单引号和双引号都可以 email = "zhangsan@example.com" # 字符串里包含变量（f-string，很常用！） message = f"你好，{name}！欢迎来到{company}" print(message) # 输出：你好，张三！欢迎来到半亩花田 # 多行字符串（用三个引号） report = """ 本月销售汇总： - 华东区：100万 - 华北区：80万 总计：180万 """ # 字符串常用操作 text = " Hello World " print(text.strip()) # 去除两端空格：Hello World print(text.upper()) # 全大写： HELLO WORLD print(text.lower()) # 全小写： hello world print(text.replace("Hello", "你好")) # 替换： 你好 World print(text.split(" ")) # 分割成列表 print(len(text)) # 长度（含空格）：15 # 判断是否包含 print("Hello" in text) # True print(text.startswith("Hello")) # False（因为有空格） print(text.strip().startswith("Hello")) # True
2

数字（int / float）—— 存数值

python
# int：整数 age = 28 count = 100 year = 2024 # float：小数 salary = 20000.0 tax_rate = 0.13 # 数学运算 print(10 + 3) # 加：13 print(10 - 3) # 减：7 print(10 * 3) # 乘：30 print(10 / 3) # 除：3.3333...（结果是float） print(10 // 3) # 整除：3（取整数部分） print(10 % 3) # 取余：1 print(2 ** 10) # 幂运算：1024 # 实用场景：计算税后工资 gross_salary = 20000 tax_amount = gross_salary * 0.13 net_salary = gross_salary - tax_amount print(f"税前：{gross_salary}元，税额：{tax_amount:.2f}元，税后：{net_salary:.2f}元") # 输出：税前：20000元，税额：2600.00元，税后：17400.00元 # 类型转换 s = "100" # 字符串"100" n = int(s) # 转为整数 100 f_num = float(s) # 转为小数 100.0 back = str(n) # 整数转回字符串 "100"
3

列表（list）—— 存多个数据，有序

列表就像一个队列，里面可以放任意数量的数据，用方括号 [] 表示。

python
# 创建列表 employees = ["张三", "李四", "王五", "赵六"] scores = [95, 87, 92, 78] mixed = ["张三", 28, True, 3.14] # 可以混合类型 # 访问元素（索引从0开始！） print(employees[0]) # 第1个：张三 print(employees[1]) # 第2个：李四 print(employees[-1]) # 最后一个：赵六 print(employees[-2]) # 倒数第二：王五 # 切片（取一段） print(employees[1:3]) # 第2到第3个：['李四', '王五'] print(employees[:2]) # 前2个：['张三', '李四'] print(employees[2:]) # 从第3个到末尾：['王五', '赵六'] # 常用操作 employees.append("钱七") # 末尾添加 employees.insert(0, "总监") # 在开头插入 employees.remove("李四") # 删除指定元素 employees.pop() # 弹出最后一个 print(len(employees)) # 列表长度 print("张三" in employees) # 判断是否包含：True # 列表排序 numbers = [3, 1, 4, 1, 5, 9] numbers.sort() # 原地排序：[1, 1, 3, 4, 5, 9] numbers.sort(reverse=True) # 倒序：[9, 5, 4, 3, 1, 1] sorted_nums = sorted(numbers) # 返回新列表，原列表不变
4

字典（dict）—— 存键值对，最常用！

字典就像真实的字典：你给一个"词"（key），它给你"解释"（value）。在数据处理中极其常用，API返回的JSON数据本质上就是字典。

python
# 创建字典 user = { "name": "张三", "age": 28, "department": "财务部", "salary": 15000, "active": True } # 访问值 print(user["name"]) # 张三 print(user.get("age")) # 28（推荐用get，不存在返回None不报错） print(user.get("phone", "未填写")) # 未填写（设置默认值） # 修改和新增 user["salary"] = 18000 # 修改已有字段 user["email"] = "zs@company.com" # 新增字段 # 遍历字典 for key, value in user.items(): print(f" {key}: {value}") # 只遍历键或值 for key in user.keys(): # 遍历所有键 print(key) for val in user.values(): # 遍历所有值 print(val) # 实战：API返回的JSON数据就是字典 api_response = { "code": 200, "message": "success", "data": { "total": 100, "orders": [ {"id": 1, "amount": 500}, {"id": 2, "amount": 300} ] } } # 提取嵌套数据 orders = api_response["data"]["orders"] for order in orders: print(f"订单{order['id']}: {order['amount']}元")

🔀 1.2 条件判断（if / elif / else）

🚦

条件判断就是"选择不同的路"

就像红绿灯：绿灯走，红灯停，黄灯等。程序根据条件决定执行哪段代码。

1

基本语法

python
# ===== 基本结构 ===== score = 85 if score >= 90: print("优秀") elif score >= 80: print("良好") # ← 这行会执行 elif score >= 60: print("及格") else: print("不及格") # ===== 比较运算符 ===== # == 等于（注意：是两个等号！一个等号是赋值） # != 不等于 # > 大于 # < 小于 # >= 大于等于 # <= 小于等于 # ===== 逻辑运算符 ===== age = 28 salary = 15000 # and：两个条件都要满足 if age >= 25 and salary >= 10000: print("符合条件") # or：任一条件满足即可 if age < 18 or age > 60: print("不在工作年龄") # not：取反 is_active = False if not is_active: print("账号未激活") # ===== 实战场景：处理订单状态 ===== def handle_order(order): status = order.get("status") amount = order.get("amount", 0) if status == "pending": if amount > 10000: print(f"大额订单{order['id']}，需要主管审批") else: print(f"订单{order['id']}自动审批通过") elif status == "approved": print(f"订单{order['id']}开始执行") elif status in ["cancelled", "rejected"]: print(f"订单{order['id']}已取消/拒绝，归档处理") else: print(f"订单{order['id']}状态未知：{status}")
单个等号 = 是赋值，双等号 == 才是比较！初学者最常犯的错误。

🔄 1.3 循环（for / while）

1

for 循环 —— 遍历已知集合

for循环用于遍历一个序列（列表、字符串、字典等），每次取出一个元素处理。

python
# ===== 基本for循环 ===== fruits = ["苹果", "香蕉", "橙子"] for fruit in fruits: print(f"我喜欢吃{fruit}") # ===== range() 生成数字序列 ===== for i in range(5): # 0,1,2,3,4 print(f"第{i+1}次循环") for i in range(1, 11): # 1到10 print(i) for i in range(0, 100, 10): # 0,10,20...90（步长10） print(i) # ===== enumerate：同时获取索引和值 ===== employees = ["张三", "李四", "王五"] for idx, name in enumerate(employees, start=1): print(f"第{idx}号员工：{name}") # ===== 嵌套循环 ===== departments = ["财务部", "运营部"] months = ["1月", "2月", "3月"] for dept in departments: for month in months: print(f"生成{dept}{month}报表") # ===== break 和 continue ===== numbers = [1, 2, 3, -1, 5, 6] for num in numbers: if num < 0: print(f"遇到负数{num}，停止处理") break # 退出整个循环 print(num) for num in numbers: if num < 0: continue # 跳过本次，继续下一个 print(num) # 只打印正数 # ===== 实战：批量处理数据 ===== raw_data = [ {"name": "张三", "amount": "1000"}, # 注意amount是字符串 {"name": "李四", "amount": ""}, # 空值 {"name": "王五", "amount": "2500"}, ] total = 0 error_count = 0 for row in raw_data: if not row["amount"]: # 空值跳过 error_count += 1 continue total += int(row["amount"]) print(f"合计：{total}元，{error_count}条数据缺失")
2

while 循环 —— 条件满足就一直执行

python
# 基本while循环 count = 0 while count < 5: print(f"第{count+1}次") count += 1 # ===== 实战：带重试的API请求 ===== max_retries = 3 attempt = 0 success = False while attempt < max_retries and not success: attempt += 1 print(f"第{attempt}次尝试...") try: # response = requests.get(url, timeout=5) success = True print("✅ 请求成功") except Exception as e: print(f"❌ 第{attempt}次失败：{e}") if attempt < max_retries: print("等待2秒后重试...") # time.sleep(2) if not success: print("❌ 所有重试均失败，请人工处理")
while循环一定要确保退出条件，否则会造成死循环！程序会一直跑停不下来。

📦 1.4 函数（def）

🔧

函数是什么？

函数就是把一段有特定功能的代码打包起来，起个名字，需要的时候就"调用"它。
好处：代码可以复用，修改一处就全部生效，而不用到处找同样的代码改。

1

函数定义与调用

python
# 最简单的函数（无参数，无返回值） def say_hello(): print("你好！") say_hello() # 调用函数：你好！ # 带参数的函数 def greet(name, company): print(f"你好，{name}！欢迎加入{company}！") greet("张三", "半亩花田") # 位置参数 greet(company="半亩花田", name="张三") # 关键字参数（顺序可以乱） # 有返回值的函数（return） def calculate_tax(salary, rate=0.13): # rate有默认值 tax = salary * rate net = salary - tax return net, tax # 可以返回多个值（实际是一个元组） net_pay, tax_amount = calculate_tax(20000) print(f"税后：{net_pay}，税额：{tax_amount}") net_pay2, _ = calculate_tax(15000, 0.1) # 用_忽略不需要的返回值 # 函数文档字符串（好习惯！） def clean_amount(value): """ 清洗金额数据：去除空格、逗号，转为浮点数 参数: value: 原始金额字符串，如 "1,234.56" 或 " 1000 " 返回: float: 清洗后的金额数值，出错返回 0.0 """ try: return float(str(value).replace(",", "").strip()) except: return 0.0 # 测试 print(clean_amount("1,234.56")) # 1234.56 print(clean_amount(" 500 ")) # 500.0 print(clean_amount("abc")) # 0.0

⚠️ 1.5 异常处理（try/except）

🚨 为什么异常处理对自动化脚本极其重要？

自动化脚本24小时无人值守地跑。网络断了、网页改版了、数据格式变了……任何意外都会让程序崩溃。
没有异常处理 = 程序崩了什么都不知道；有异常处理 = 出问题了自动记日志、发告警、继续处理其他数据。

1

完整的异常处理语法

python
# ===== 完整结构 ===== try: # 可能出错的代码放这里 result = 10 / 0 # 这会引发 ZeroDivisionError except ZeroDivisionError as e: print(f"除零错误：{e}") # 捕获特定异常 except ValueError as e: print(f"值错误：{e}") # 捕获另一种异常 except Exception as e: print(f"未知错误：{e}") # 捕获所有异常（兜底） else: print("没有发生任何异常，执行这里") # 可选 finally: print("无论如何都会执行这里") # 通常用于关闭连接、释放资源 # ===== 常见异常类型 ===== # FileNotFoundError 文件不存在 # KeyError 字典键不存在 # IndexError 列表索引越界 # ValueError 值类型不对（如int("abc")） # ConnectionError 网络连接失败 # TimeoutError 超时 # ZeroDivisionError 除以零 # ===== 实战：健壮的数据处理函数 ===== import logging logger = logging.getLogger("data_processor") def safe_process_row(row): """安全地处理一行数据，出错不崩溃""" try: # 各种可能出错的操作 name = row["name"].strip() # 可能KeyError amount = float(row["amount"]) # 可能ValueError result = amount * 1.13 # 计算含税金额 return {"name": name, "amount_with_tax": result, "status": "ok"} except KeyError as e: logger.warning(f"字段缺失：{e}，数据：{row}") return {"status": "error", "reason": f"字段缺失:{e}"} except ValueError as e: logger.warning(f"数据格式错误：{e}，数据：{row}") return {"status": "error", "reason": f"格式错误:{e}"} # 批量处理，出错的跳过，继续处理后面的 data_list = [ {"name": "张三", "amount": "1000"}, {"name": "李四", "amount": "abc"}, # 这个会出错 {"name": "王五", "amount": "2000"}, ] results = [safe_process_row(row) for row in data_list] success_count = sum(1 for r in results if r["status"] == "ok") print(f"成功：{success_count}条，失败：{len(results)-success_count}条")
