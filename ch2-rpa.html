🤖 2.0 RPA 到底是什么？

💡

一个真实的例子理解RPA

公司财务每天要把OA系统里的报销申请，一条一条手动复制粘贴到ERP系统。一天要重复这个动作200次，每次3分钟，一天600分钟，10小时！

有了RPA：让软件机器人来做这件事。它会自动打开OA系统 → 找到待处理申请 → 读取数据 → 打开ERP系统 → 填入数据 → 提交 → 循环。全程无需人工，每条30秒，200条100分钟，下班前全做完。

✅ RPA适合做什么？

  • 重复性高、规则固定的操作
  • 多系统之间的数据搬运
  • 表单填写、数据录入
  • 定时报表生成和发送
  • 网页数据采集
  • 文件格式转换和处理

❌ RPA不适合做什么？

  • 需要复杂判断的业务决策
  • 界面经常大改的系统
  • 需要创意或理解语义的任务
  • 实时性要求极高的场景

🤖 AI Agent解决了什么？

  • 能理解非结构化输入（自然语言）
  • 能应对界面变化（视觉理解）
  • 能自主规划多步骤任务
  • 能处理异常情况，不那么"脆"

📌 元素定位方法（最重要！）

什么是元素定位？

要让机器人点击某个按钮，首先得找到这个按钮。就像告诉别人"点网页上写着'登录'的那个蓝色按钮"。
在代码里，用XPath或CSS选择器来精确定位元素。

python
from selenium import webdriver from selenium.webdriver.common.by import By from selenium.webdriver.support.ui import WebDriverWait from selenium.webdriver.support import expected_conditions as EC from selenium.webdriver.chrome.options import Options # ===== 启动浏览器 ===== options = Options() # options.add_argument("--headless") # 无头模式，不显示浏览器窗口（后台运行） driver = webdriver.Chrome(options=options) driver.maximize_window() # 最大化窗口 # ===== 定位元素的6种方式 ===== # 1. 通过ID定位（最精准，优先用） elem = driver.find_element(By.ID, "username") # 2. 通过CLASS_NAME定位 elem = driver.find_element(By.CLASS_NAME, "btn-submit") # 3. 通过NAME属性定位 elem = driver.find_element(By.NAME, "password") # 4. 通过CSS选择器（灵活，常用） elem = driver.find_element(By.CSS_SELECTOR, "#login-form input.email") elem = driver.find_element(By.CSS_SELECTOR, "button[type='submit']") # 5. 通过XPATH（功能最强，可以根据文字定位） elem = driver.find_element(By.XPATH, "//button[@type='submit']") elem = driver.find_element(By.XPATH, "//span[text()='登录']") # 根据文字找 elem = driver.find_element(By.XPATH, "//span[contains(text(),'登')]") # 模糊匹配 # 6. 通过LINK_TEXT（专门找链接） elem = driver.find_element(By.LINK_TEXT, "忘记密码") # ===== 如何获取XPath？===== # 在Chrome中：右键元素 → 检查 → 在开发者工具中右键对应HTML → Copy → Copy XPath

🏆 完整案例：自动登录并抓取数据

python
from selenium import webdriver from selenium.webdriver.common.by import By from selenium.webdriver.support.ui import WebDriverWait from selenium.webdriver.support import expected_conditions as EC import pandas as pd import time def scrape_order_system(): """登录订单系统，抓取今日订单数据""" driver = webdriver.Chrome() wait = WebDriverWait(driver, 15) try: # 1. 打开登录页 driver.get("https://oa.company.com/login") # 2. 等待用户名输入框出现 username_input = wait.until( EC.presence_of_element_located((By.ID, "username")) ) # 3. 输入账号密码 username_input.send_keys("admin") driver.find_element(By.ID, "password").send_keys("password123") driver.find_element(By.XPATH, "//button[text()='登录']").click() # 4. 等待登录成功（等待跳转后的元素出现） wait.until(EC.presence_of_element_located((By.CLASS_NAME, "nav-menu"))) print("✅ 登录成功") # 5. 导航到订单列表页 driver.get("https://oa.company.com/orders?date=today") # 6. 等待表格加载 wait.until(EC.presence_of_element_located((By.CSS_SELECTOR, "table.order-table"))) # 7. 抓取所有行 rows = driver.find_elements(By.CSS_SELECTOR, "table.order-table tbody tr") data = [] for row in rows: cells = row.find_elements(By.TAG_NAME, "td") if len(cells) >= 4: data.append({ "订单号": cells[0].text, "客户": cells[1].text, "金额": cells[2].text, "状态": cells[3].text, }) print(f"✅ 抓取到 {len(data)} 条订单") # 8. 保存到Excel df = pd.DataFrame(data) df.to_excel("今日订单.xlsx", index=False) print("✅ 已保存到 今日订单.xlsx") return data except Exception as e: driver.save_screenshot("error_screenshot.png") # 出错时截图留证 print(f"❌ 发生错误：{e}") raise finally: driver.quit() # 无论成功失败都关闭浏览器 # 执行 scrape_order_system()

🔌 2.3 API 自动化（requests库详解）

🍽️

API就是系统间的"服务窗口"

你去餐厅（系统A）点菜，服务员（API）把你的需求告诉厨房（系统B），厨房做好了通过服务员返回给你。
你不需要知道厨房怎么做菜，只需要按菜单下单（按API文档调用）即可。

📡 HTTP请求方法

方法用途类比
GET获取数据（查询）查看菜单
POST提交数据（新增）下订单
PUT完整更新数据换一道菜
PATCH部分更新数据加辣少盐
DELETE删除数据取消订单
python
import requests import json # ===== GET请求：获取数据 ===== response = requests.get( url="https://api.example.com/orders", params={"page": 1, "size": 50, "status": "pending"}, # URL参数 headers={ "Authorization": "Bearer eyJhbGci...", # 认证Token "Content-Type": "application/json" }, timeout=10 # 超时10秒 ) # 检查请求是否成功 print(response.status_code) # 200=成功, 401=未授权, 404=不存在, 500=服务器错误 if response.status_code == 200: data = response.json() # 解析JSON响应 print(data) else: print(f"请求失败：{response.status_code} - {response.text}") # ===== POST请求：提交数据 ===== payload = { "order_id": "ORD-2024-001", "action": "approve", "approver": "张三", "comment": "审批通过" } response = requests.post( url="https://api.example.com/orders/approve", json=payload, # 自动序列化为JSON并设置Content-Type headers={"Authorization": "Bearer token"} ) result = response.json() if result.get("code") == 0: print("✅ 审批提交成功") else: print(f"❌ 提交失败：{result.get('message')}") # ===== 带重试的健壮请求函数 ===== import time def robust_request(method, url, max_retry=3, **kwargs): """带自动重试的HTTP请求""" for attempt in range(max_retry): try: resp = requests.request(method, url, timeout=10, **kwargs) resp.raise_for_status() # 非200状态码抛出异常 return resp except requests.exceptions.Timeout: print(f"第{attempt+1}次请求超时") except requests.exceptions.ConnectionError: print(f"第{attempt+1}次连接失败") except requests.exceptions.HTTPError as e: print(f"HTTP错误：{e}") break # HTTP错误不重试 if attempt < max_retry - 1: time.sleep(2 ** attempt) # 指数退避：1s, 2s, 4s... raise Exception(f"请求{url}失败，已重试{max_retry}次")

🏗️ 标准流程架构

""" 标准自动化任务模板 - 可以复用的框架结构 每个新项目都按这个结构写，养成好习惯 """ import logging import os from datetime import datetime from dotenv import load_dotenv # ===== 1. 配置管理 ===== load_dotenv() # 从.env文件加载敏感配置 # 配置参数（不要把密码写死在代码里！） CONFIG = { "db_host": os.getenv("DB_HOST", "localhost"), "db_password": os.getenv("DB_PASSWORD"), "api_token": os.getenv("API_TOKEN"), "output_dir": "./output", } # ===== 2. 日志配置 ===== os.makedirs("logs", exist_ok=True) logging.basicConfig( level=logging.INFO, format='%(asctime)s [%(levelname)s] %(funcName)s - %(message)s', handlers=[ logging.FileHandler(f"logs/{datetime.now().strftime('%Y%m%d')}.log", encoding='utf-8'), logging.StreamHandler() ] ) logger = logging.getLogger(__name__) # ===== 3. 核心业务函数 ===== def fetch_data(): """数据采集""" logger.info("开始采集数据...") # 实际采集逻辑 return [] def process_data(raw_data): """数据处理""" logger.info(f"开始处理 {len(raw_data)} 条数据...") # 实际处理逻辑 return [] def save_result(processed_data): """保存结果""" logger.info("保存结果...") # 写入数据库/文件 def notify(success_count, error_count): """发送通知""" msg = f"任务完成！成功:{success_count}条，失败:{error_count}条" logger.info(msg) # send_ding_talk(msg) # 发钉钉通知 # ===== 4. 主入口 ===== def main(): start_time = datetime.now() logger.info(f"===== 任务开始 =====") try: raw_data = fetch_data() processed = process_data(raw_data) save_result(processed) notify(len(processed), 0) except Exception as e: logger.error(f"任务失败：{e}", exc_info=True) notify(0, -1) finally: elapsed = (datetime.now() - start_time).seconds logger.info(f"===== 任务结束，耗时{elapsed}秒 =====") if __name__ == "__main__": main()
